# 🎉 2026 Holiday Calendar for Toddlers

A colorful, interactive calendar that makes every holiday an adventure! Built for little ones who love tapping things and watching confetti explode. 🎊

![Preview](preview-with-images.png)

## ✨ Features

- 🗓️ **27 holidays** — US majors plus fun ones like Dinosaur Day 🦕 and Lego Day 🧱
- 🎆 **Confetti modals** — tap a holiday, get a party!
- 🖼️ **Cute clipart** for every single holiday (with emoji fallbacks)
- 📱 **Month navigation** — swipe through the whole year
- 🫧 **Animated emojis** floating around because why not
- 🔤 **Fun fonts** via Google Fonts

## 🚀 How to Use

Just open `index.html` in any browser. That's it. No install, no build step, no npm, no nothing.

```
open index.html
```

Hand the device to your toddler. Stand back.

## 🛠️ Tech Notes

- **Single HTML file** — everything lives in `index.html`
- **Zero dependencies** — no frameworks, no build tools
- **Images** in the `images/` folder (clipart for all 27 holidays)
- Google Fonts loaded via CDN (only external resource)
